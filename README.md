# Champion Project

Проект выполнен для чемпионата по верстке. Включает верстку макета с SCSS, оптимизацией графики и JS-анимациями.

## Установка зависимостей
Установите все зависимости командой:

```bash
npm ci
```

## Запуск в режиме разработки
Для запуска dev-сервера используйте:

```bash
npm start
```

## Сборка продакшн-версии
Чтобы собрать проект для продакшн:

```bash
npm run build
```

## Линтинг
Проверка кода на ошибки и соответствие стандартам:

```bash
npm run lint
```

Для проверки BEM-классов:

```bash
npm run lint-bem
```

## Структура проекта
```
source/
├── sass/      # SCSS стили
├── js/        # JavaScript
├── img/       # Изображения
├── fonts/     # Шрифты
├── public/    # Статические файлы
└── index.html
dist/          # Сборка продакшн
```

## Особенности
- SCSS с переменными, миксинами и reset
- Dev-сервер с автооткрытием браузера и горячей перезагрузкой
- Проверка кода на соответствие стандартам, включая BEM-классы (через npm run lint-bem)
- Статические файлы из public/ копируются в сборку как есть
